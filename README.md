# Supernovas_Excelrs-Medical-Crew
This is a web application developed during DECTHON using web dev + crewAI + Python
The project is about an AI powered virtual medical assistant which helps with patient's queries, provides advice for the symptoms and will help to consult to doctor if necessary.
This project mainly helps in connecting patients and healthcare providers with the help of crewAI and Python.
